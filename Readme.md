# SpendSense

## Overview

This project is a simple Finance Dashboard built using HTML, CSS, and JavaScript.
It allows users to track income, expenses, view transactions, and see basic spending insights.

## Features

* Financial Summary (Balance, Income, Expense)
* Transactions Table
* Search and Filter Transactions
* Role-Based UI (Viewer / Admin)
* Insights (Highest Spending Category)
* Responsive Design

## Role Functionality

* **Viewer:** Can only view transactions and summary
* **Admin:** Can add new transactions

## Tech Used

* HTML
* CSS
* JavaScript

## How to Run

Open `index.html` in your browser.
